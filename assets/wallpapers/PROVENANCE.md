# Wallpaper provenance

The repository's root `LICENSE` does **not** override the licenses listed here.

retrieved on 2026-07-11 from the linked Wikimedia Commons file pages.

Every bundled image is a genuine 6000×4000-pixel photograph. Nine files are
byte-for-byte copies of 6000×4000 originals. The other seven were downsampled
and minimally center-cropped from larger originals; no wallpaper was upscaled.
